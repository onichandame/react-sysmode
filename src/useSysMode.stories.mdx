# useSysMode

A hook retrieving system modes.

## Dark

Obtains dark/light mode from system. Captures changes.

Windows 10:

`Settings > Personalization > Colors > Choose your color`

Gnome 3:

`Tweak > Theme`

## Basic

<Preview>
  <Story name="Basic">
    {
      () => {
        const { dark } = useSysMode()
        return (
          <div>
            {dark ? 'dark' : 'light'}
          </div>
        )
      }
    }
